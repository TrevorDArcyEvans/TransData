# TransData - Transforming Your Data Across Platforms


## Pre-requisites
*   .NET SDK 9


## Getting started
```bash
# clone repo
git clone https://github.com/TrevorDArcyEvans/TransData.git

# build
cd TransData/src
dotnet build

# run app
cd TransData.App
dotnet run
```
